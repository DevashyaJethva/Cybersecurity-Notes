#  TCP (Transmission Control Protocol)

TCP is a Transport layer protocol designed for reliability and accuracy.

It establishes a stable connection between two devices before data transfer.

---

#  Key Features

- connection-oriented
- error checking
- guaranteed delivery
- correct data order
- flow control

---

#  Advantages
- accurate delivery
- error correction
- ensures correct order

#  Disadvantages
- slower than UDP
- requires stable connection
- retransmission delays communication

---

#  Where TCP is Used

TCP is commonly used in:

- file transfers
- email communication
- web browsing

These applications require complete data delivery.
